## Getting Started

### Creating an Account

1. Click the **"Sign Up"** button in the top navigation
2. Fill in your information:
   - First Name
   - Last Name
   - Username (unique identifier)
   - Email Address
   - Password (minimum 8 characters)
3. Click **"Create Account"**
4. Check your email for a verification link
5. Click the verification link to activate your account

**Alternative:** You can also sign up using Google or Facebook OAuth.

### Logging In

1. Click **"Login"** in the top navigation
2. Enter your email and password
3. Click **"Login"**

**Note:** If you haven't verified your email, you'll be prompted to do so.

### Forgot Password?

1. Click **"Forgot Password?"** on the login page
2. Enter your email address
3. Check your email for a password reset link
4. Click the link and set a new password

## Browsing Courses

### Viewing All Courses

1. Click **"Courses"** in the navigation menu
2. Browse through available courses
3. Use the search bar to find specific courses
4. Filter by category or difficulty level

### Course Details

Each course card shows:
- Course title and description
- Difficulty level (Beginner, Intermediate, Advanced)
- Category
- Course image (if available)

Click on a course card to view:
- Full course description
- Course modules and lessons
- Enrollment button

## Enrolling in Courses

### How to Enroll

1. Browse to the course you want to take
2. Click **"Enroll Now"** button
3. You're automatically enrolled! No approval needed.

### Viewing Enrolled Courses

1. Go to your **Dashboard**
2. Your enrolled courses appear in the "My Courses" section
3. Each course shows:
   - Progress bar
   - Completed lessons count
   - "Continue Learning" button

### Unenrolling

1. Go to the course page
2. Click **"Unenroll"** button
3. Confirm your action

## Learning Experience

### Accessing Lessons

1. Go to your Dashboard
2. Click **"Continue Learning"** on an enrolled course
3. Or browse to the course and click on a module
4. Click on a lesson to start learning

### Lesson Features

Each lesson includes:
- **Learning Objectives**: What you'll learn
- **Estimated Time**: How long it takes
- **Rich Content**: Formatted text, images, and code examples
- **Code Examples**: Syntax-highlighted code snippets
- **Exercises**: Practice problems

### Navigation

- **Previous/Next Buttons**: Navigate between lessons
- **Course Sidebar**: Quick access to all lessons in the course
- **Breadcrumbs**: See where you are in the course structure

### Completing Lessons

1. Read through the lesson content
2. Review code examples
3. Complete any exercises
4. Click **"Mark as Complete"** when finished

**Note:** Time spent on lessons is automatically tracked.

## Bookmarks

### Bookmarking Lessons

1. While viewing a lesson, click the **"Bookmark"** button
2. The button changes to show it's bookmarked
3. Click again to remove bookmark

### Accessing Bookmarks

1. Go to your Dashboard
2. Scroll to the "Bookmarked Lessons" section
3. Click on any bookmarked lesson to access it

## Exercises

### Completing Exercises

1. Navigate to a lesson with exercises
2. Read the exercise description
3. Review any hints provided
4. Write your code in the code editor
5. Click **"Submit Code"**

### Exercise Feedback

- You'll receive immediate feedback
- Correct solutions mark the lesson as complete
- You can resubmit if needed

## Troubleshooting

### Email Not Received

- Check your spam/junk folder
- Verify the email address is correct
- Click "Resend Verification" if needed
- Wait a few minutes (emails may be delayed)

### Can't Access Lesson

- Ensure you're enrolled in the course
- Check that the lesson is published
- Try refreshing the page
- Clear your browser cache

### Progress Not Updating

- Make sure you clicked "Mark as Complete"
- Refresh the page
- Check your internet connection
- Contact support if issue persists

### Exercise Not Submitting

- Check that you've entered code
- Verify your internet connection
- Try refreshing the page
- Check browser console for errors

### Mobile Issues

- Use a modern mobile browser (Chrome, Safari)
- Ensure JavaScript is enabled
- Try landscape orientation for better viewing
- Clear browser cache if needed
